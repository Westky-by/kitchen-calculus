## เพิ่มช่วงวันที่ (Date Range) ในหน้าเอกสารทั้งหมด

เพิ่มตัวกรอง **ช่วงวันที่** รูปแบบ `DDMMYYYY-DDMMYYYY` ที่หน้ารายการเอกสารทั้งหมดของ Tax Invoice (แท็บ "เอกสารทั้งหมด") เพื่อใช้ค้นหาเอกสารตามช่วงวันที่ ทำงานร่วมกับช่องค้นหา keyword เดิม

### สิ่งที่จะเพิ่ม (UI)

- แถบตัวกรองใหม่ด้านบนตารางเอกสารทั้งหมด (อยู่แถวเดียว/ต่อจากช่องค้นหาปัจจุบัน):
  - ปุ่มพรีเซ็ต: `วันนี้` `7 วัน` `เดือนนี้` `ปีนี้` `กำหนดเอง`
  - ช่อง **จากวันที่** และ **ถึงวันที่** (ใช้ `<input type="date">` เพื่อให้เลือกง่าย)
  - ช่องแสดงผลรูปแบบ `DDMMYYYY-DDMMYYYY` (read-only badge) เพื่อให้ผู้ใช้เห็นชัดเจนตามที่ร้องขอ
  - ปุ่ม **ล้างช่วงวันที่**

### พฤติกรรมการกรอง

- ถ้ากำหนดช่วงวันที่: กรอง `doc_date` ให้อยู่ระหว่าง from–to (รวมปลายทั้งสอง)
- ทำงานร่วมกับช่องค้นหาเดิม (เลขที่ / ลูกค้า / ผู้สร้าง / สถานะ / ยอด) แบบ AND
- ถ้าไม่กำหนด: แสดงทั้งหมดเหมือนเดิม
- ค่าเริ่มต้น: ไม่มีช่วง (ว่าง)

### ไฟล์ที่แก้

- `src/pages/TaxInvoice.tsx` เท่านั้น (เฉพาะแท็บ "เอกสารทั้งหมด")
  - เพิ่ม state: `dateFrom`, `dateTo`, `datePreset`
  - เพิ่ม UI ช่วงวันที่เหนือ/ข้างช่องค้นหาปัจจุบัน
  - ปรับ logic filter ของ list ให้เช็ค `doc_date` ในช่วงที่เลือก
  - ไม่แก้ไข query ฝั่ง Supabase (กรองฝั่ง client จาก dataset เดิมที่โหลดมาแล้ว) เพื่อลดผลกระทบ

### ไม่แตะ

- แท็บ Dashboard สรุป, ฟอร์มสร้าง/แก้ไขเอกสาร, PDF/พิมพ์, และ business logic อื่น ๆ
